# Task Tracking: zt2_index

创建Pickles ZT2平台的统一索引页面，作为中央导航中心和演示门户。该页面需要展示完整的6步用户旅程，从报价生成到配送完成，包括客户端演示和运营监督功能。页面应该提供专业的品牌展示，具备响应式设计，支持移动端(430px)和桌面端的内容展示。
